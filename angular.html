Angular 1.X
AngularJS is the first version of the framework developed by Google, which was released in 2010
Advantages
  • Two-way data binding is a powerful feature
  • If a change happens to this object, it’s updated in the view
  • If the view changes values, it’s updated on the $scope object, as well as all other view references
Disadvantages
  • high performance cost
  • Inability to compose many NG-apps on the same page. This can cause name clashes
Angular 2.X
The release of Angular 2 generated numerous changes to the initial framework, as it was rewritten in TypeScript and it supported mobile.
Advantages
  • Angular 2 encourages using unidirectional data flow.
  • Angular 2 is Easier.TypeScript allows code optimization using the OOPS concept.
  • It is mobile-oriented.
Disadvantages
  • Migrating the application to Angular 2 required too many modifications due to different syntax
  • Application size reduction and syntax changes made it impossible to simply upgrade from AngularJS to the new version
Angular 4.X
In Angular 4.X, production bundle size reduced by 60%.It enables a fast development process.
Advantages
  • It enables a fast development process.
  • Full TypeScript support helps in building bulky applications.
  • Tests are easy to write.
Disadvantages
  • It’s slow when displaying enormous amounts of data.
Angular 5.X
The release of fifth version concentrated on optimizing Angular CLI and compiler work.
Advantages
  • Simpler Progressive Web Applications
  • Material Design
  • Build optimizer
  • Angular Universal API and DOM
Disadvantages
  • Seems a bit difficult to switch from one version to another like it was in previous cases
Angular 6.X
Angular 6 was released on May 4, 2018. It includes the Component Development Kit (CDK), Angular CLI (Command Line Interface), and Angular Material Package Update
Advantages
  • TypeScript 2.7+ support
  • Stable Angular Material with CDK ( Component Development Kit)
  • Ivy renderer
  • Bazel Compiler
  • Addition of navigationSource and restoredState to NavigationStart
Disadvantages
  • limited SEO options and poor accessibility for search engine crawlers.
  • Difficult to learn
Angular 7.X
This new release of the web application development framework has bought certain attractive features that made this framework unique as compared to its previous versions
Advantages
  • CLI Prompts
  • Enhanced Application Performance
  • Improved accessibility for selects
Disadvantages
  • Not supported for Node 10
  • Slower than Angular 8
Angular 8.X
The Angular community has released Angular 8 with an impressive list of changes and improvements including the much awaited Ivy compiler as an opt-in feature.
Advantages
  • Support TypeScript 3.4
  • Supports Web Workers
  • Preview of Ivy available
  • Lazy loading
Disadvantages
  • There is no precise manual and extensive, all-inclusive documentation
  • Steep learning curve
  • Third-party integration is complicated.
Angular 9.X
With Angular9, all the existing drawbacks have been modified. Lot of factors have affected the previous versions of Angular.
Advantages
  • Amazing Debugging
  • Faster Performance
  • Better Testing
  • Completely new CSS class and Style Binding
Disadvantages
  • Decline in popularity
  • Lot of boilerplate code
Angular 10.X
Angular 10 provides a stricter project setup for making a new workspace using ng new.
Advantages
  • Optional Stricter Settings
  • New Features ie: Date Range Picker
  • Enhanced Community Engagement
  • Boost in ngcc Performance
  • Removals and Corrections
Disadvantages
  • Angular 10 does not contain the positive jsdoc remarks from the NPM for assisting the advanced optimization of a closure compiler.
Angular 11.X
The major features of this version are router performance improvements, automatic inlining of fonts and stricter types.
Advantages
  • Updated Hot Module Replacement (HMR) Support
  • Updates on Operation Byelog
  • Automatic Inlining of Fonts
  • Faster Builds
  • Webpack 5 Support
Disadvantages
  • Angular 11 has dropped the support for TypeScript 3.9 and only supports TypeScript 4.0
  • DisAdv2
  • DisAdv3
Angular 12.X
Angular 12 is smaller, faster, and easier to use. It will be making Angular developer's life easier.
Advantages
  • Better Developer Ergonomics @angular/forms
  • Better developer ergonomics with strict typing for @angular/forms
  • Reducing Framework Overheard by Removing Legacy
  • Leverage full framework capabilities with Zone.js opt-out
  • Improved build performance with ngc as a tsc plugin distribution
Disadvantages
  • Deprecating Support for IE11
